Автоматизация Отчетов VK Ads

Добро пожаловать на домашнюю страницу приложения **"Автоматизация Отчетов VK Ads"** – вашего незаменимого инструмента для эффективного анализа рекламных кампаний.

Наше приложение создано для автоматизации сбора, обработки и наглядного представления ключевых метрик из ваших рекламных кабинетов (VK Ads, TargetHunter) прямо в Google Таблицах. Это решение идеально подходит для маркетологов, SMM-специалистов и аналитиков, стремящихся сэкономить время и повысить точность отчетности.

Основные функции приложения:

  • **Автоматический сбор данных:** Приложение самостоятельно запрашивает и импортирует данные о рекламных кампаниях из подключенных источников.
  • **Генерация детализированных отчетов:** Создает ежедневные, еженедельные и ежемесячные отчеты в Google Таблицах.
  • **Расчет ключевых метрик:** Автоматически вычисляет расходы, показы, переходы, CTR, eCPC, лиды, конверсию в лид, подписчиков Senler и стоимость подписчика.
  • **Визуализация данных:** Строит интерактивные диаграммы, показывающие распределение лидов по дням недели и часам суток.
  • **Интеграция с Telegram-ботом:** Для проверки статуса подписки и получения персонализированных уведомлений о работе приложения.
  • **Простая инициализация:** Позволяет быстро создать копию шаблонной таблицы со всеми необходимыми скриптами на вашем Google Диске.

Как приложение использует ваши данные:

Мы ценим вашу конфиденциальность. Приложение запрашивает минимально необходимые разрешения Google для обеспечения своей основной функциональности.

  • **Доступ к Google Таблицам (`https://www.googleapis.com/auth/spreadsheets`):** Это основная цель приложения. Мы читаем данные для агрегации (из листа "День", "Бюджет") и создаем, записываем, обновляем и форматируем отчеты, формулы и диаграммы в листах ("День/Час", "Отчёт неделя").
  • **Доступ к Google Диску (`https://www.googleapis.com/auth/drive`):** Используется для создания копии шаблонной таблицы на вашем Google Диске при первом запуске приложения. Это необходимо для начала работы.
  • **Подключение к внешним службам (`https://www.googleapis.com/auth/script.external_request`):** Позволяет приложению безопасно взаимодействовать с API TargetHunter и BotHunter для проверки вашей подписки и отправки персонализированных уведомлений в Telegram.
  • **Управление свойствами документа (`https://www.googleapis.com/auth/script.properties`):** Мы используем это для безопасного хранения ваших API-токенов (для VK Ads, TargetHunter) и других настроек автоматизации внутри самой Google Таблицы.
  • **Просмотр вашего адреса электронной почты (`https://www.googleapis.com/auth/userinfo.email`):** Ваш email используется только для добавления его в название скопированной таблицы ("Копия Отчет (ваш@email.com)") для удобства идентификации.

Мы не продаем, не передаем и не используем ваши данные в рекламных целях или для любых других целей, не связанных с прямой функциональностью этого приложения.
