Политика конфиденциальности

Добро пожаловать на домашнюю страницу приложения "Автоматизация Отчетов VK Ads" – вашего незаменимого инструмента для эффективного анализа рекламных кампаний.

Наше приложение создано для автоматизации сбора, обработки и наглядного представления ключевых метрик из ваших рекламных кабинетов (VK Ads, TargetHunter) прямо в Google Таблицах. Это решение идеально подходит для маркетологов, SMM-специалистов и аналитиков, стремящихся сэкономить время и повысить точность отчетности.

Основные функции приложения:

  • Автоматический сбор данных: Приложение самостоятельно запрашивает и импортирует данные о рекламных кампаниях из подключенных источников.
  • Генерация детализированных отчетов: Создает ежедневные, еженедельные и ежемесячные отчеты в Google Таблицах.
  • Расчет ключевых метрик: Автоматически вычисляет расходы, показы, переходы, CTR, eCPC, лиды, конверсию в лид, подписчиков Senler и стоимость подписчика.
  • Визуализация данных: Строит интерактивные диаграммы, показывающие распределение лидов по дням недели и часам суток.
  • Интеграция с Telegram-ботом: Для проверки статуса подписки и получения персонализированных уведомлений о работе приложения (через введенный вами ID чат-бота).
  • Простая инициализация: Позволяет быстро создать копию шаблонной таблицы со всеми необходимыми скриптами на вашем Google Диске.

Как приложение использует ваши данные и разрешения Google:

Мы ценим вашу конфиденциальность и запрашиваем только минимально необходимые разрешения Google для обеспечения основной функциональности приложения. Каждое разрешение используется строго по назначению:

  • Просмотр и изменение таблиц, в которых установлено это приложение (`https://www.googleapis.com/auth/spreadsheets.currentonly`):

    Это основное разрешение, позволяющее Приложению читать данные (например, из листов "День", "Бюджет"), а также создавать, записывать, обновлять и форматировать отчеты, формулы и диаграммы в листах ("День/Час", "Отчёт неделя") **только в той Google Таблице, в которой оно было установлено**. Приложение не имеет доступа к другим таблицам или файлам на вашем Google Диске. **Это разрешение также включает доступ к свойствам документа (Document Properties), которые используются для безопасного хранения API-токенов и других настроек, необходимых для автоматической работы приложения.**

  • Подключение к внешней службе (`https://www.googleapis.com/auth/script.external_request`):

    Позволяет приложению безопасно взаимодействовать с внешними API (например, VK Ads API, TargetHunter API) для получения актуальных данных о рекламных кампаниях и лидах. Эти данные затем используются для заполнения вашей таблицы.

  • Разрешить этому приложению запускаться, когда вас нет (`https://www.googleapis.com/auth/script.scriptapp`):

    Используется для создания и управления автоматическими триггерами (например, для ежедневного обновления отчетов по расписанию), чтобы обеспечить актуальность ваших данных без ручного вмешательства.

  • Отображение и запуск стороннего веб-контента в подсказках и боковых панелях в приложениях Google (`https://www.googleapis.com/auth/script.container.ui`):

    Позволяет приложению создавать интерактивные элементы интерфейса (такие как пользовательские меню или боковые панели) внутри Google Таблицы для более удобного управления функциями.

  • Просмотр вашего адреса электронной почты (`https://www.googleapis.com/auth/userinfo.email`):

    Ваш адрес электронной почты используется **исключительно** для автоматического добавления его в название скопированной таблицы (например, "Копия Отчет (ваш@email.com)") для удобства идентификации и управления вашими файлами. Мы не используем ваш email для маркетинговых рассылок или обмена с третьими сторонами.

Важно: Приложение "Автоматизация Отчетов VK Ads" **НЕ СОХРАНЯЕТ** ваши данные на своих серверах. Все конфиденциальные данные, такие как API-токены (для VK Ads, TargetHunter), ваш Telegram UID и URL вашей таблицы, сохраняются **непосредственно в свойствах той Google Таблицы (Document Properties)**, которую вы используете с приложением. Это необходимо для обеспечения непрерывной и автоматической работы фоновых функций (например, ежечасных запросов к API) и персонализированных уведомлений через Telegram-бота.

Мы не продаем, не передаем и не используем ваши данные в рекламных целях или для любых других целей, не связанных с прямой функциональностью этого приложения. Все данные обрабатываются в соответствии с нашей Политикой конфиденциальности.
